## What I Do

I specialize in **rescuing struggling SaaS products** and **modernizing legacy PHP codebases** — the kind of work most engineers avoid. If your product is slow, unmaintainable, or stuck in deployment hell, that's my domain.

**Core expertise:**
- 🔥 **SaaS Rescue** — taking over broken products, stabilizing them, shipping what was promised
- ⚙️ **DevOps & Infrastructure** — zero-downtime deployments, containerization, cloud migration
- 🏗️ **Laravel / PHP** — clean architecture, API design, performance optimization
- ☁️ **AWS** — EC2, RDS, ECS, S3, Route53, auto-scaling
- 🐳 **Docker & CI/CD** — GitHub Actions, automated pipelines, staging → production workflows

> *Nearly a decade in the industry. Based in Belgrade, Serbia.*
